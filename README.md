  <div style="padding: 20px;">
    <h2>📌 Project Overview</h2>
    <p>
      The <b>HR Analytics Project</b> focuses on analyzing employee-related data to help organizations 
      make better decisions regarding <b>attrition</b>, <b>performance</b>, <b>salary trends</b>, and <b>employee satisfaction</b>.  
      Using the <code>HR_Analytics.csv</code> dataset, we explore insights and visualize important HR metrics.
    </p>

    <!-- Dataset Section -->
    <h2>📂 Dataset Information</h2>
    <p>
      The dataset <b>HR_Analytics.csv</b> contains details about employees, including:
    </p>
    <ul>
      <li>Employee ID & Demographics</li>
      <li>Department & Job Roles</li>
      <li>Monthly Income & Experience</li>
      <li>Attrition (Yes/No)</li>
      <li>Performance Ratings & Work-life Balance</li>
    </ul>

    <!-- Features -->
    <h2>🚀 Features & Insights</h2>
    <ul>
      <li>Employee attrition analysis</li>
      <li>Department-wise performance comparison</li>
      <li>Salary distribution & promotion patterns</li>
      <li>Work-life balance visualization</li>
      <li>Predictive analytics for retention</li>
    </ul>

    <!-- Installation -->
    <h2>⚙️ Installation & Usage</h2>

      </li>

        <pre><code>cd hr-analytics</code></pre>
      </li>
      <li>Install required libraries:
        <pre><code>pip install -r requirements.txt</code></pre>
      </li>
      <li>Run the analysis:
        <pre><code>python hr_analysis.py</code></pre>
      </li>
    </ol>
